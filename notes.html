Client/server

Definition: Client-server model is a type of software model consisting of two types of applications, called clients and servers. Clients access services or resources made available by servers.

HTML tags

HTML tags are used to dictate to the browser how it must display the HTML file. HTML tags are displayed within < > brackets. To close a tag, add the same tag with a forward slash at the end of the command </ >.
Some tags do not need to be closed. For example, "< br >", which is used to introduce a line break, does not need to be closed.
Tags should be nested in one another, that is, if you start a tag within another tag, then it should terminated before the encasing tag as well.

Lab 2: HTML Knowledge

What is an HTML tag? Give three examples of tags

HTML tags are keywords that define the structure of a webpage, and instruct the browser on how to format and display the page.
Three examples of tags are: <HTML>, <head>, <body>.

What is an empty tag? How does it differ from a regular tag?

Empty tags differ from regular tags in the sense that they do not need to enclosed. For example, <br> is a standalone tag that does not need to be closed.

What is an attribute of an HTML tag?

Attributes provide additional information about HTML elements. They are added in the opening tag. For example, "lang" is an attribute that is used in the <HTML> tag to declare its language.

What is the correct way for HTML tags to nest?

If a tag is opened inside another tag, then it cannot be closed after the tag encumbering it has been closed. This would result in an error. The tag must be closed before the enclosing tag is closed.
